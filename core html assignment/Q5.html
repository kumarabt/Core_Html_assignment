VARIOUS TECH STACKS OF FULL STACK WEB DEVELOPMENT

HTML
HTML (HyperText Markup Language): The backbone of web content, used for structuring web pages.
CSS
CSS (Cascading Style Sheets): Used for styling and layout, to control the visual presentation of web pages.
JAVASCRIPT
A versatile scripting language that enables interactivity and dynamic behavior in web applications. 
 Popular JavaScript libraries and frameworks include:

            React.js: Developed by Facebook, it's a popular library for building user interfaces.
            Angular: A framework by Google for building dynamic web applications.
            Vue.js: A progressive JavaScript framework for building user interfaces.
