VARIOUS HTML TAGS

  1. HEADING TAG:heading tags in html are used to signify importance of the text written in a line.
    there are various heading tags according to their importance, namely h1 tag,h2 tag and so on upto h6 tag.
  2. BOLD or B TAG:this tag is used to display text in bold letters.
  3. EM OR I TAG:em or emmit tag and i tag are both used to convert text into italic font.
    but em tag also emphasis on the text in seo.
  4. PARAGRAPH TAG:paragraph tag is used to write and render specific peice of content in paragraph format in web page.
  5. PRE TAG:pre tag is like paragraph tag but it renders content in as it is format as written in html file.
  6. META TAG:meta tag is always included in head tag.
    it usually contains information about informatin or important information about webpage that is not rendered specifically on web page.
